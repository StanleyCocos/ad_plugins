## flutter常用模块封装

### 简介
1. app_info 获取设备信息
2. auth_manager 权限管理
3. global_notification 单例广播
4. open_manager 外部功能启动(拨打电话，打开设置等) 未完善
5. regex_manager 常用正则判断
6. screenshot_manager 屏幕截屏管理
